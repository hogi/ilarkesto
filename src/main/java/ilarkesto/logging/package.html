Usage

  • Declare your logger as a final static constant in your class. This guarantees only one initialization, for all instances of your class.
  • Use the logging methods with arbitrary parameters. The logger will format them.
The logger trys to interpret the parameters given to a logging method such as ILogger.debug(Object[]). If it can not, then it calls the toString() -method.

Examples

public class MyClass {

	private static final Logger	LOG	= Logger.get(MyClass.class);

	public static void main(String[] args) {
		// default log
		LOG.debug("started MyClass with following arguments:",args);
		
		// log fatal error
		LOG.error("failed to start MyClass",new RuntimeException("Error"));
	}

}
